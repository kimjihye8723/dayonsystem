TD2000   G 3   Quick Configuration Guide   FW1.1  3  Introduction  The TD2000 G 3   is an advanced deep learning 3D sensor designed to set new standards in traffic counting with unparalleled accuracy  and a comprehensive feature set. It is ideal for top - view mounting and excels in both traffic counting and comprehensive behavioral  analyti cs based on   traffic counting .   Compared to its previous generations like TD2000,   the TD2000 G3 boasts exceptional AI  capabilities. When   working   with the TD4000 series of edge computing units, it supports   Precise Count with the   advanced Re - ID  functionalities   that filter out   inflated   counts, such as staff, repeat visitors, couriers, and brief visitors, ensuring precise customer data.  It provides versatile staff exclusion methods: 1) a convenient Staff Exclusion Button for easy implementation, 2) embedded de ep  learning - based uniform identification that requires no additional devices or tags, saving operational costs, and 3) precise Re - I D staff  identification.   Please refer to the Datasheet of the product should you need more details on the features that can be enabled by using  the device.  The   TD   sensor technical documentation suite includes:  •   Quick Start   Guide  •   Quick   Configuration Guide (this document)  •   TD   Remote Control Server   User Guide  About this Document  The   TD 2000   G 3   3D   Intelligent   Sensor   Quick Configuration Guide   details basic configuration instructions for IT departments and  implementation engineers   to set up the device properly for operating in most cases . For advanced configurations, contact our  technical representatives for assistance. TD2000   G 3   Quick Configuration Guide   FW1.1  15  Major Stream  This   tab   (Major Stream)   allows you to   configure the video parameters for   the main stream of the camera . Currently, the sensor  supports   a single stream encoded by H264   only .  Note that for drop - down parameters like   Enco ding Profile ,   GOP Mode,   Rate Control Mode   and   Resolution ,   after   modification,  you need to click the   [Save]   button to complete the modification.   For slider configuration parameters like   Frame Rate ,   Max Bitrate ,  you don’t need to   click   [Save]   as   the modification takes effect   immediately.  The parameters are described as follows:  Payload :   H.264   is a high - performance video codec technology, with high compression ratio, flexible processing, ultra - low bit rate,  saving network transmission bandwidth and storage space.  Encoding   Profile :   the higher the encoding   level   at the same bitrate, the higher the image quality .   The   Baseline ,   Main profile , and  High profile   are increasing   in encoding level in turn.  GOP   (Group of Pictures) :   the number of frames between two keyframes. The larger   the GOP value , the smaller the stream   and  the   lower   the image quality;   Conversely,   the smaller   the GOP value, the   larger the stream   and the higher   the image quality.  Rate Control Mode :  V BR   stands   for   variable   bit   rate,   and   CBR   stands   for   constant   bit   rate.   CBR   keeps   the   bitrate   the   same   throughout   the   encoding   pr  ocess , which may cause video   mosaic.   VBR   adjusts   bitrate   (not exceeding the max bitrate) during encoding process. Its  compression speed is relatively slow, but it can   retain   a better   image   qualit y and clarity in complex scenes.  F rame   R ate : the number of frames per second of the video.   The higher the value, the better of the image quality   requiring more  bandwidth and storage space.  Max   B itrate:   i ndicates the upper limit of the encoding bitrate, if the   bitrate is too low, the image quality will deteriorate.  Please set   parameters   according to the actual bandwidth situation.  The recommended parameter is configured as:   GOP   set as   an integer multiple of the frame rate; When the resolution is  1920x1080, the   max   bitrate is recommended at 4096kbps ;   and the max bitrate is recommended at 2048kbps when the resolution  is 1280x720. The   max bitr ate should be configured with reference to the actual situation of resolution, picture quality, bandwidth,  etc., and it is not recommended to set it   larger than necessary .
